# 명지대학교 부동산대학원 공인중개사 석사과정 모집 페이지 (MVP)

중장년층도 쉽게 볼 수 있도록 **큰 글자**, **단순한 화면**, **모바일/PC 반응형**으로 만든 간단한 모집 페이지입니다.

## 화면 구성

- `index.html` : 과정 소개 + 큰 **온라인 지원하기** 버튼
- `apply.html` : 온라인 지원서 입력 폼
- `success.html` : 제출 완료 화면 ("지원서가 정상 접수되었습니다")
- `styles.css` : 공통 스타일

> 중요: 본 과정은 **공인중개사 자격 보유자만 지원 가능**하도록 안내하고 있습니다.

## 1) 아주 쉬운 실행 방법 (로컬)

아래 순서대로 하면 컴퓨터 초보자도 바로 열 수 있습니다.

### 방법 A: 파일 더블클릭 (가장 쉬움)
1. 이 폴더를 바탕화면처럼 찾기 쉬운 위치에 둡니다.
2. 폴더를 열고 `index.html` 파일을 찾습니다.
3. `index.html` 파일을 **더블클릭**합니다.
4. 인터넷 브라우저(크롬/엣지)에서 페이지가 열리면 성공입니다.
5. **온라인 지원하기** 버튼 → 지원서 작성 → 제출 완료 화면까지 확인합니다.

> 만약 더블클릭해도 안 열리면: `index.html` 파일 우클릭 → **연결 프로그램** → 크롬(또는 엣지) 선택

### 방법 B: VS Code Live Server 사용
1. VS Code에서 이 폴더를 엽니다.
2. `index.html` 우클릭 → **Open with Live Server** 클릭
3. 자동으로 브라우저가 열리면 사용하면 됩니다.

## 2) 입력 항목

지원서 폼에는 아래 항목이 포함되어 있습니다.

- 성명
- 휴대전화
- 이메일
- 생년월일
- 소속 중개사무소 또는 회사명
- 공인중개사 자격 여부
- 관련 경력
- 지원동기
- 자기소개
- 지원서 파일 업로드
- 개인정보 수집 및 이용 동의 체크박스 (필수)

## 3) Vercel 배포 (초보자용)

이 프로젝트는 정적(Static) 페이지라 Vercel에 바로 올리기 쉽습니다.

1. GitHub에 이 프로젝트를 업로드합니다.
2. [Vercel](https://vercel.com/)에 로그인합니다.
3. **Add New → Project** 선택
4. GitHub 저장소를 선택하고 **Deploy** 클릭
5. 배포 완료 후 제공된 URL로 접속합니다.

`vercel.json`이 포함되어 있어 URL을 깔끔하게 유지하기 좋습니다.

## 4) 참고

- MVP(최소 기능 제품) 구조로 구성했습니다.
- 로그인/회원가입 기능은 포함하지 않았습니다.
- 현재 폼 제출은 데모용으로 `success.html` 이동 방식입니다.
